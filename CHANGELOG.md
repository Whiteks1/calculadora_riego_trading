# Changelog

## 1.0.0 - 2026-03-27

Primera version estable del proyecto como `pre-trade risk workbench`.

- calculadora web de riesgo con validacion de setup, fees y slippage
- escenarios, historico persistente y exportacion CSV
- trade plan determinista en JSON y CSV
- CLI headless para generar artifacts sin depender del navegador
- motor C++ con pruebas de paridad frente al core en JavaScript
- mini backtester visual con equity curve
- CI de contrato/paridad y pruebas UI basicas
- deploy en GitHub Pages y metadata publica del repo alineada para portfolio
